# GoTHStack
Example GoTH Stack Project

### Notes
* If you are building backend applications using Golang it is nice to be able to add a front end without having to add the complexity of JS
 * HTMX is a common way of doing this
  * `golang` + `HTMX` + `templ` is the goth stack

### templ
 * Allows you to create components that render fragmented HTML + compose them into pages documents or apps
 * contains server side rendering static rendering
 * compile templ componets into to go code
 * they have a specific structure you have to follow to use them
